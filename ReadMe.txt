В функции fillStacks заполняются два стека случайными числами. Функция outputSeparately выводит содержимое двух стеков отдельно. Функция mergeStacks объединяет содержимое двух стеков в отсортированный set, а затем переносит его в новый стек. В main используется бесконечный цикл, который позволяет пользователю выбирать функции для заполнения, просмотра и объединения двух стеков, а также просмотра отсортированного стека и завершения программы.
